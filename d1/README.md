# how to use

1. download or clone the project
2. create `.env` file and setup you openai_api_key
3. run the project with (dotenv will load your apikey from your .env file)
```bash
dotenv -- promptfoo eval
```
4. visualize the result on your browser with
```bash
dotenv -- promptfoo view
```
5. go to: http://localhost:15500/eval
6. you can modify the prompt and re-run the eval to evaluate your prompt
7. enjoy!!
